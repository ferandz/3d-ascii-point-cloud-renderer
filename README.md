# Spinning vertices of cube in C 🧊

Вращающийся куб на C - маленький проект, который был создан как эксперимент, вдохновленный видео на YouTube канале [Code Fiction](https://www.youtube.com/watch?v=p09i_hoFdd0)

Цель эксперимента была создать свою реализацию вращающихся куба/вершин куба с глубиной, а также разобраться в линейной алгебре.

Внимание! проект был создан чисто как эксперимент и не претендует на хороший код и/или хорошую реализацию.

## Особенности
- **Использование структур:** В отличии от варианта, показанного на канале Code Fiction, моя реализация включает использование структур с координатами точек
- **Подключение ANSI escape-последовательностей:** изменяется режим консоли через WinAPI для гарантированной работы ANSI escape-последовательностей в Windows
- **Динамический шейдинг по глубине:** символы вершин (`#`, `*`, `.`) рассчитываются автоматически в зависимости от глубины
- **Универсальность:** можно отрисовывать не только вершины куба, но и вершины других фигур, изменяя массив точек

 ## Стек
 * Язык программирования: C (C99 стандарт)
 * Библиотеки: стандартные (`stdio.h`, `math.h`, `string.h`, `stdint.h`), а также системные библиотеки `unistd.h` и `windows.h`
